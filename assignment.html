Python Data Types

Python has several built-in data types:

  • Integer: a whole number, positive or negative, without decimals.
  • Float: a number with a decimal point.
  • Boolean: a data type that represents the concept of true or false.
  • String: a sequence of characters that can contain letters, numbers, and symbols.
  • List: an ordered collection of items, which can be of different types.
  • Tuple: an ordered collection of items, which cannot be modified.
  • Set: an unordered collection of unique items.
  • Dictionary: a collection of key-value pairs.
Python Dictionaries and Calculator Example

Python Dictionaries

A dictionary is an unordered collection of key-value pairs. In Python, dictionaries are defined using curly braces {} with each key-value pair separated by a colon. For example:

		my_dict = {
			"name": "John",
			"age": 30,
			"city": "New York"
		}

You can access the values in a dictionary by referencing the keys. For example, to get the value associated with the "name" key in the above dictionary, you would use:

		name = my_dict["name"]

Python dictionaries are commonly used to store and access data in a way that is efficient and easy to read.

Python Calculator Example

A calculator is a tool that performs mathematical operations. In Python, you can create a calculator using functions and operators.

Example: Simple Calculator

This example shows a simple calculator that allows users to enter two numbers and choose an operation to perform.

		def add(x, y):
			return x + y

		def subtract(x, y):
			return x - y

		def multiply(x, y):
			return x * y

		def divide(x, y):
			return x / y

		print("Select operation.")
		print("1.Add")
		print("2.Subtract")
		print("3.Multiply")
		print("4.Divide")

		choice = input("Enter choice (1/2/3/4):")

		num1 = float(input("Enter first number: "))
		num2 = float(input("Enter second number: "))

		if choice == '1':
			print(num1,"+",num2,"=", add(num1,num2))

		elif choice == '2':
			print(num1,"-",num2,"=", subtract(num1,num2))

		elif choice == '3':
			print(num1,"*",num2,"=", multiply(num1,num2))

		elif choice == '4':
			print(num1,"/",num2,"=", divide(num1,num2))
		else:
			print("Invalid input")

This calculator allows users to enter two numbers and choose an operation to perform from a list of options. It then uses functions to perform the selected operation and prints the result to the console.
